# TripleMatch3D
 *Remake of "Triple match 3D" game!*
 

 **Tech stack:** Zenject, Event System, Audio, Scene Management, Game loop, Unity UI, Game Levels, Game Mechanics, TextMeshPro, Using asset store.

 **Platform:** Android, PC
